Compétences développées:

  • Meilleure connaissance du langage C# ( Csharp ) et de l’IDE Visual Studio
  • J’ai progressé et développé mes compétences en programmation orientée objet.
  • Maîtrise de l’utilisation de collections : List , Set et Map (Dictionary en C#).
  • Organisation d’un projet à partir d’un Trello.
  • Amélioration de ma façon de faire des tests unitaires.

Le projet consiste à développer une application en C# avec une interface graphique, en réponse à un besoin exprimé par un client. Cette application est un jeu de plateau nommé Hyperstellar se déroulant dans l’espace, elle possède des variantes grâce à de nombreuses extensions disponibles. Les étapes du projet comprennent l’analyse du besoin, la conception , l'implémentation et les tests.

Afin d’assurer une bonne mise en place du projet mon équipe et moi-même avons réalisé un Trello ( outil de gestion de projet ) , un diagramme UML et un dépôt Git qui permet de mettre notre code en commun.
